# Get entire igv browser image in svg

Get entire igv browser image in svg

## Usage

``` r
# S4 method for class 'igvR'
saveToSVG(obj, filename)
```

## Arguments

- obj:

  An object of class igvR

- filename:

  character string, the name of the file to which the svg text will be
  written

## Value

A character vector
